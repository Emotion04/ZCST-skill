# 珠海科技学院图书馆

> 最后更新: 2026-06-06
> 信息来源: 珠海科技学院图书馆官网、学校官方推文

## 基本信息

- **官网**：https://library.zcst.edu.cn
- **位置**：校园中心区域

## 开放时间

> ⚠️ 具体开放时间可能随学期和节假日调整，请以图书馆官网公告为准。

- **阅览区**：通常 8:00 - 22:00
- **自习室**：通常 7:00 - 22:30
- **借还书服务**：工作时间内开放
- 期末复习期间可能延长开放时间

## 借阅规则

- **借书数量**：本科生一般可借 10-15 册
- **借期**：通常 30-60 天，可续借
- **逾期**：超期将产生罚款，具体标准见图书馆公告
- **还书**：阅览区一楼设自助借还机

> 详细借阅规则以图书馆官网 [读者及图书借阅管理办法](https://library.zcst.edu.cn/2022/0628/c3541a168615/page.htm) 为准。

## 楼层功能

| 楼层 | 功能区 |
|------|--------|
| 一楼 | 阅览区、自助借还机、图书杀菌机 |
| 二楼 | 理科阅览区、自习区、书库 |
| 三楼 | 理科阅览区、自习区、书库 |
| 四楼 | 需从三楼右侧小门前往，文科阅览区、自习区、书库 |

## 设施与服务

| 设施 | 说明 |
|------|------|
| 自助借还机 | 阅览区一楼 |
| 图书自助杀菌机 | 阅览区一楼，30 秒清除 99.5% 细菌 |
| 自助打印机 | 图书馆行政区一楼（可打印成绩单、学籍表、就读证明等） |
| 直饮水机 | 公共区域免费使用 |
| 自习座位 | 各楼层均设自习区 |

## 座位预约

> ⚠️ 具体预约方式请以图书馆最新公告为准。一般通过图书馆微信公众号或「我的珠科」APP 预约。

## 小贴士

- 期末复习期间座位紧张，建议提前预约
- 图书馆空调较足，建议带件薄外套
- 图书馆提供免费 WiFi
