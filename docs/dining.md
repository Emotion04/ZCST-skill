# 珠海科技学院餐饮指南

> 最后更新: 2026-06-05
> 信息来源: 小红书高赞帖子整理（@上课爱抠手、@千山暮雪、@珠科球知道 等）

## 校内食堂

### 一饭（旧饭）

- **位置**: 校内（已搬到中厅）
- **特点**: 传统食堂，价格实惠
- **推荐**: 煲仔饭、干蒸
- **注意事项**: 部分档口有变动，建议到现场看最新情况

### 二饭（新饭）

- **位置**: 校内新区域
- **特点**: 品类较多，人流量大
- **推荐**:
  - **炊班主** — 多人推荐，味道好
  - **螺蛳粉** — 被称为"世界上最好吃的螺蛳粉"（夸张但多人认可）
- **注意事项**: 高峰期排队较长

### 三饭（粤港食堂）

- **位置**: 校内
- **特点**: 份量大，价格适中
- **推荐**: 各类粤菜、煲仔饭
- **评价**: "份量大这一块 粤港还是太超前了"

## 校内餐饮店

### 一品鲜

- **位置**: 校内
- **特点**: 多人推荐的热门店铺
- **推荐**: 各类粉面
- **评价**: "一天想吃两顿一品鲜"

### 汇轩

- **位置**: 旧饭区域（已搬到中厅）
- **特点**: 校内茶餐厅
- **推荐**: 港式茶点、奶茶
- **注意事项**: 2025年有评论说变成KFC了，建议确认最新情况

### 奈斯蛋糕

- **位置**: 校内
- **特点**: 蛋糕甜品店
- **推荐**: 蛋糕（奶油好评）
- **评价**: "蛋糕好吃，面包一般般"

## 校外餐饮（南门/安堂商业街）

### 南门美食街

- **位置**: 学校南门外
- **特点**: 小吃种类丰富，价格亲民
- **推荐**:
  - **苕皮** — "贼好吃"
  - **臭豆腐** — "每次点爆辣 巨无敌香"
  - **沙姜鸡** — 多人推荐
- **评价**: 新生必逛的美食区域

### 安堂商业街

- **位置**: 学校附近
- **特点**: 各类小吃、餐饮店
- **推荐**: 根据个人口味探索

## 周边推荐餐厅

### 老家春饼（东北菜）

- **位置**: 学校附近
- **人均**: 中等
- **推荐**: 东北菜、春饼
- **评价**: "除了等位久没啥缺点"
- **注意事项**: 高峰期需要等位

### 桑拿鸡

- **位置**: 学校附近
- **推荐**: 桑拿鸡
- **评价**: "推荐！还有一家东北菜也强烈推荐"

### 老珠人气烧烤

- **位置**: 学校附近
- **特点**: 烧烤店
- **评价**: 小红书美食排行榜有提及

## 实用信息

### 支付方式

- **饭卡**: 校内主要支付方式
- **充值**: 可在食堂充值点或线上充值
- **微信/支付宝**: 部分店铺支持

### 外卖

- **平台**: 美团、饿了么
- **配送**: 可送至校内
- **校内取餐点**: 各宿舍楼下

### 避雷提示

- **养生煲煲菜**: 2025年9月已关
- **烧肉妹冰粉**: 2025年3月已关
- 部分店铺可能有变动，建议到现场确认
