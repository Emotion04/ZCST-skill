# 珠海科技学院宿舍指南

> 最后更新: 2026-06-06
> 信息来源: 学校官方信息、新生攻略推文

## 宿舍规格

- **房型**：统一四人间，上床下桌
- **床铺尺寸**：1.9m × 0.9m
- **园区**：竹园、康园、松园、桂园、梅园、榕园（共 42 栋，6 层/栋，回廊式设计）

## 硬件设施

| 设施 | 说明 |
|------|------|
| 空调 | 每间 1 台，可制冷 |
| 风扇 | 每间 1 台 |
| 独立卫浴 | 每间宿舍独立卫生间 + 淋浴间 |
| 阳台 | 独立阳台 |
| 饮水机 | 康园、松园在走廊设直饮水机，其余园区每间配独立饮水机 |
| 书柜/衣柜 | 每人一套，多层收纳 |
| 网络接口 | 有，可自行办理校园卡开通宽带 |

## 热水

- **热水时间**：18:00 - 23:00（一般时段） 23:00开始变小

## 门禁与断电

- **门禁时间**：23:00，人脸识别刷卡进出
- **断电规则**：
  - 夏季（5-10 月）不断电
  - 冬季 23:00（周末 23:30）至次日 6:00 断电，仅风扇可用。（2022年开始停止实行。
  - 空调线路不受断电影响
- **电器功率限制**：理论 ≤ 1200W，严禁使用电饭锅、电磁炉、热得快等大功率电器

## 公共设施

| 设施 | 说明 |
|------|------|
| 外卖柜 | 每栋宿舍楼架空层，共 43 套 3092 格口，30 分钟保温，公众号「爽提」取餐 |
| 洗衣机 | 宿舍楼下公共区域 |
| 直饮水机 | 教学楼、实验楼、图书馆、体育馆等公共区域免费使用 |

## 住宿费

- **普通本科生**：3000 元/生·学年（空调四人间，不含水电费）
- 水电费由宿舍成员分摊自理

## 水电费缴纳

- **水费**：宿舍平均完后，个人支付宝绑定的校园卡中缴纳
- **电费**：「我的珠科」APP 充值，民电。消耗多的日子一天20度，一般一天3-4度

## 小贴士

- 床品购买建议：学校床铺尺寸特殊（90cm × 190cm），按实际尺寸购买
- 珠海潮湿，建议备除湿盒/除湿剂
- 蚊帐建议选全包款，防蚊防虫效果更好
- 遮光窗帘建议 1.2m-1.4m 高度即可
