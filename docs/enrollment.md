# 珠海科技学院新生入学指南

> 最后更新: 2026-06-05
> 信息来源: 珠海科技学院新媒体中心官方推文（2025-08-10）

## 证件及相关资料

### 必带证件

1. **新生录取通知书**（报到必备）
2. **高考准考证**（若录取通知书丢失时可用，建议带上备用）
3. **个人档案**
4. **党团组织关系资料**
   - 具体党团组织转接操作，请根据《2025级新生入学须知》要求进行
   - 党组织关系转接：广东省内在党务系统内电子转接，不开具纸质版介绍信；广东省外开具纸质版介绍信
5. **身份证原件及复印件**
6. **兵役证原件及复印件**（仅2025年12月31日前年满18周岁且不超过24周岁的广东省户籍适龄男性公民需要携带）
7. **户口本复印件**（户主页和本人页）
8. **近期免冠半身白底彩色证件照**（1寸、2寸各4张，电子版免冠照（用于学生卡片、学生登记表、学生证等办理））

### 其他资料

- **需要迁移户口的同学**：[户口迁移证]
- **申请助学贷款等国家和学校资助项目的同学**：[家庭经济困难学生认定资料]
  - 广东省家庭经济困难学生认定申请表
  - 广东省家庭经济困难学生认定分析表

### Tips

1. 如果是个人携带档案的，入学后按要求统一上交；若个人档案是由高中邮寄到校的，则不需要个人携带
2. 报到时证件、资料较多，可以准备一个文件袋收纳，可以避免材料散落、丢失
3. 学校毗邻澳门，若想去澳门旅游，记得提前办理好并带上港澳通行证哦~

## 衣物用品

### 衣物清单

1. 衣服
2. 腰带（军训期间可能用到哦）
3. 袜子
4. 内衣裤
5. 拖鞋
6. 鞋子（建议双以上，便于换穿）
7. 衣架
8. 晾衣夹、晾盘（用于夹袜子等）
9. 大夹子（盖被子的时候会用到）
10. 洗衣袋（洗衣机洗滚筒的衣服时使用，和内衣一起洗可防止衣服磨损或变形）

### 衣物携带建议

1. 非省内同学回家较为方便，暂时穿不到的衣物可以不带
2. 离家远的同学最好提前备好衣物和相关生活物品哦~

### 穿衣建议

1. **2-3月份**：天气多雾潮湿，需要加件薄外套
2. **4月份**：天气回暖，5月份气温明显升高，可以逐渐换上短袖
3. **6-9月份**：天气偏热，穿着基本以短袖为主，出门记得做好防晒，可戴帽子、穿防晒衫；9月份天气依旧炎热，晚间气温略降，夜晚出门可穿个薄外套，避免着凉
4. **10月份**：天气有所转凉，出门可带件外套，冬天的持续时间较短，大概11月份才逐渐转冷，到12月份才用得上毛衣
5. **1、2月份**：风大，天气较为寒冷，少不了用风衣、棉袄、厚外套等防风外套进行保暖御寒

## 床上用品

### 床品清单

1. 被子+被套（被子建议薄厚各一张，即空调被+棉被）
2. 床单
3. 枕头枕套
4. 蚊帐（拉开门，怕蚊虫的同学可以选择全包款）
5. 学生床垫（床的规格：90CM*190CM）
6. 凉席（可根据个人实际需要购买）

### 床品携带建议

- 珠海10月中下旬气温仍较高，11月稍有转凉，但温度仍偏高，北方的同学可以先带一床空调被，天气转凉时再让父母把棉被寄来学校

## 提升住宿质量系列

1. 舒适好看的睡衣
2. 遮光窗帘（适合需要一定私人空间的同学，1.2米-1.4米高即可）
3. 眼罩、耳塞
4. 床上小桌子（宅人福音）
5. 收纳盒
6. 坐垫

## 日常用品

### 清洁用品

1. 纸巾（抽纸/卷纸&手帕纸）
2. 棉签
3. 牙刷/牙线
4. 抹布（开学时和日常用于清洁宿舍）

### 生活用品

1. 防蚊喷雾（建议选择含避蚊胺成分的产品）
2. 挂钩粘钩（可以贴在墙上的）
3. 小镜子（可以放在宿舍桌面用的）
4. 筷子/勺子
5. 饭盒/碗（可以用来装水果等食物）
6. 水杯（用于宿舍）、水瓶/保温杯（可随身携带的）
7. 钥匙扣（方便保管好多把钥匙）
8. 各种规格的收纳盒
9. 除湿盒/除湿剂
10. 吹风机（1000w以下）
11. 雨伞（必备！要足够牢固！）
12. 运动用品（学校的所有运动场地均免费开放，新生可以提前准备好喜欢的运动用品！）

## 物品邮寄

- **邮寄地址**：广东省珠海市金湾区三灶镇安基东路8号珠海科技学院
- 衣架、晾衣夹、拖鞋这些物品，可以选择在学校超市购买或者网购邮寄到学校，这样可以给你的行李箱留出更多空间哦~
