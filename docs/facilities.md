# 珠海科技学院校园设施

> 最后更新: 2026-06-06
> 信息来源: 学校官方推文、新生攻略

## 快递站

快递站较为齐全，靠近南二门。
**学校快递地址**：广东省珠海市金湾区三灶镇珠海科技学院（广东省珠海市金湾区安东基路8号）
**邮政编码**：519041

| 快递公司 | 位置 |
|---------|------|
| 中通、申通、极兔 | 校内快递中心（南二门附近，校园巴士站与音乐舞蹈学院之间） |
| 圆通 | 综合楼一楼（妈妈驿站） |
| 韵达、邮政 | 珠科哆哆代收点 |
| 京东 | 校内菜鸟驿站 |
| 顺丰 | 校外（以最新通知为准） |
| 邮政包裹/信件 | 珠科邮政服务点（南二门右侧快递服务集中点） |
| 校外菜鸟驿站 | 南一门对面美宜佳超市内 |

> 取件码首位数字对应不同驿站，注意核对取件地址。

## 校园巴士（小蓝）

- **运营时间**：7:00 - 22:45
- **票价**：1 元/人次
- **起点**：南门
- **实时查询**：「我的珠科」小程序可查实时动态

## 共享单车（青桔）

- 遍布校园各角落
- 微信/支付宝扫码使用
- 季卡约45元，闲鱼有优惠短期卡
