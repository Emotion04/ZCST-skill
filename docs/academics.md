## 转专业指南

### 基本信息

- **报名时间**：2025年12月5日8:00-12月10日12:00
- **报名网址**：https://jw.zcst.edu.cn
- **浏览器建议**：火狐、UC、谷歌、360（极速模式）
- **用户名**：学生学号
- **初始密码**：身份证后六位

### 报名流程

1. 登录教务系统
2. 点击"报名申请" → "学生转专业申请"
3. 选定拟转入专业（可填报一个）
4. 确认点击"提交"

### 重要规则

- **转专业次数**：在校期间每位学生只有一次转专业机会
- **转出后限制**：转出后不得转回原专业
- **公示时间**：12月12日公示拟接收转专业学生名单，公示期七天
- **生效时间**：2025-2026学年第二学期开学正式转入新专业

### 申请条件

1. 具有良好的思想道德品质，学习态度端正
2. 经我校录取已取得2025级普通本科注册学籍的大一新生
3. 身体条件符合拟转入专业招生的要求
4. 对拟转入专业有一定的兴趣和专长

### 特殊限制

- **体育类、音乐类**：只能学院内专业互转，不得转入学院外其他专业
- **美术类**：只能动画、视觉传达设计、艺术与科技专业互转
- **舞蹈编导专业**：不可转入、转出
- **学前教育专业**：可以与学院外其他文理专业互转

### 不能转专业的情况

1. 招生时已有明确规定不能转专业的学生
2. 招生时有特殊要求的提前批次录取学生
3. 休学或保留学籍期间的学生
4. 其他有失公平、公正、公开原则的

### 转专业后学分认定

1. 转专业学生仍须在原专业完成第一学期课程学习及考试
2. 转入新专业前已修读的学分，凡符合转入专业人才培养方案要求的相同或相近课程及学分，经接收学院审核同意并报教务处审定后予以免修
3. 不符合转入专业人才培养方案要求的课程及学分，可作为公共选修课计入学分（合计不超过4学分）
4. 转入新专业学习的学生须自行补修转入专业人才培养方案规定的第一学期学习内容和相应学分
